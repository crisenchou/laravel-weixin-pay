# crisen: WeixinPay

**WeixinPay driver for the crisen PHP payment processing library**


## Installation

Omnipay is installed via [Composer](http://getcomposer.org/). To install, simply add it
to your `composer.json` file:

```json
{
    "require": {
        "crisen/laravel-weixinPay": "*"
    }
}
```
